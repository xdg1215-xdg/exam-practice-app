# 云龙定制

中盐常化 2026 年管理人员制度学习考题练习 App，基于 Vite + Vue 3 + Capacitor 构建。

## 功能

- 📖 顺序练习：按题目顺序逐题练习
- 🎲 随机练习：随机抽题，模拟真实考试
- ❌ 错题重练：自动收藏错题，反复练习
- 📊 答题统计：自动评分，查看错题回顾

## 题库

- 单选题：280 道
- 多选题：162 道
- 判断题：125 道
- 合计：567 道

## 技术栈

- Vite 8 + Vue 3 (Composition API)
- Capacitor 6 (Android 打包)
- localStorage 错题持久化
