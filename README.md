# 制造系统练习

中盐常化 2026 年职能部室中层管理人员制度学习考题练习 App，基于 Vite + Vue 3 + Capacitor 构建。

## 功能

- 📖 顺序练习：按题目顺序逐题练习
- 🎲 随机练习：随机抽题，模拟真实考试
- ❌ 错题重练：自动收藏错题，反复练习
- 📊 答题统计：自动评分，查看错题回顾

## 题库

- 单选题：278 道
- 多选题：109 道
- 判断题：126 道
- 合计：513 道

## 技术栈

- Vite 8.x
- Vue 3.5
- Capacitor 8.x
- Java 21 (Android 构建)

## 本地开发

```bash
npm install
npm run dev
```

## 构建 Android APK

通过 GitHub Actions 自动构建，下载 Artifacts 即可获取 `app-debug.apk`。
